# Binary Synthwave Calculator

Single-file browser prototype for the binary-only music concept.

## What it does
- Binary-only input with dedicated 1 / 0 buttons
- Futuristic synthwave calculator UI
- Presets
- Mathematical scoring:
  - coherence
  - harmonic quality
  - pulse
  - novelty
  - atmosphere
- Sequence visualizer
- Best-of-32 genome search
- Text-to-binary seed
- In-browser Web Audio playback
- Export current genome as text

## Run
Just open `binary_synthwave_calculator.html` in a browser.
